# Gyst Design Tokens

The Gyst app uses an Art Nouveau-inspired design system based on the work of Alphonse Mucha. This page documents all design tokens (colors, typography, spacing, etc.) used throughout the application.

## Color Palettes

Gyst offers 5 different Art Nouveau-inspired color themes. Use the theme toolbar in Storybook to switch between them.

### Theme Palettes

You can switch between these themes using the data-theme attribute:

- **mucha-classic**: Warm earthy tones (plum, gold, terracotta)
- **mucha-spring**: Fresh spring garden colors (rose, sage, cream)
- **mucha-autumn**: Harvest season palette (burgundy, amber, forest)
- **mucha-evening**: Evening jewel tones (sapphire, amethyst, midnight)
- **mucha-ocean**: Ocean depths palette (teal, coral, sand)

## Typography

The app uses a sophisticated typography system with custom fonts inspired by the Art Nouveau era.

### Font Families

- **Display (Headings)**: Playfair Display - ornamental serif for titles
- **Body (Text)**: Inter Variable - modern sans-serif for readability
- **Ornamental (Accents)**: Cinzel Decorative - decorative capitals

## Dark Mode

All themes support dark mode. Switch using the dark mode toolbar in Storybook.

In dark mode:
- Background colors become darker
- Text colors invert for contrast
- Primary/accent colors are slightly desaturated
- Borders become more subtle

## Accessibility

All color combinations meet WCAG 2.1 AA standards for contrast:
- Normal text: 4.5:1 minimum
- Large text: 3:1 minimum
- Interactive elements have clear focus states

## Usage

### CSS Variables

All tokens are available as CSS custom properties:

```css
.my-component {
  color: hsl(var(--primary));
  background: hsl(var(--background));
  border: 1px solid hsl(var(--border));
  border-radius: var(--radius);
  padding: var(--spacing-4);
}
```

### Tailwind Classes

Use Tailwind utility classes that map to these tokens:

```jsx
<div className="bg-background text-foreground p-4 rounded-lg shadow-md">
  <h1 className="heading-2 text-primary">Hello</h1>
  <p className="text-muted-foreground">Welcome to Gyst</p>
</div>
```

### Theme Attribute

Apply themes via the data-theme attribute:

```jsx
<div data-theme="mucha-spring">
  <!-- Spring theme colors applied -->
</div>
```
