# PWA Icons

This directory should contain the PWA app icons in various sizes.

## Required Icons

The following icon sizes are needed for the PWA to work properly:

- `icon-72x72.png` - 72x72px
- `icon-96x96.png` - 96x96px
- `icon-128x128.png` - 128x128px
- `icon-144x144.png` - 144x144px
- `icon-152x152.png` - 152x152px
- `icon-192x192.png` - 192x192px (minimum required)
- `icon-384x384.png` - 384x384px
- `icon-512x512.png` - 512x512px (recommended for splash screens)

## Design Guidelines

The icon should:
- Feature the Gyst branding
- Use Art Nouveau design elements inspired by Alphonse Mucha
- Incorporate the Mucha mauve color palette (#8B4789)
- Work well on both light and dark backgrounds
- Be recognizable at small sizes

## Generating Icons

You can use tools like:
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)

Or design in Figma/Illustrator and export at the required sizes.

## Placeholder

For development, you can create placeholder icons using any graphic design tool or online generator. The actual production icons should be created by a designer following the Art Nouveau aesthetic of the app.
